# Welcome to sidetrack!

> [!CAUTION]
> **Sidetrack is deprecated and no longer maintained (June 2026).** This repository is archived (read-only) and the npm packages (`sidetrack`, `@sidetrack/client-prisma`) are deprecated. Existing versions remain installable, but there will be no further updates.
>
> [Effect](https://effect.website/)'s workflow and cluster primitives (`@effect/workflow`, `@effect/cluster`) now cover this space. For other Postgres-backed job queues, consider [graphile-worker](https://github.com/graphile/worker) or [pg-boss](https://github.com/timgit/pg-boss).
>
> Thanks to everyone who used it and contributed. — [@aniravi24](https://github.com/aniravi24)

## Introduction

Sidetrack is a robust TypeScript-first background job processing library, backed by PostgreSQL.
